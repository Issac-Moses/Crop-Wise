### Local Execution

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Issac-Moses/Crop-Wise.git
   ```
2. **Launch Application**
   - Open `index.html` directly in your browser.
   - Recommended: Use **Live Server** or a local Python server.
   ```bash
   python -m http.server 8000
   ```

### Protocol Guidelines
1. Fill in the required **Nutrient** and **Property** fields.
2. Select the specific **Soil Type** from the dropdown menu.
3. Click **INITIATE ANALYSIS** to trigger the AI processing unit.
